# Break Fast

A game built with [Three.js](https://threejs.org/) about wanting to go on a break, fast. Can be played on [https://apmds.github.io/break-fast/](https://apmds.github.io/break-fast/)

This project is being developed as the final project for my [Introduction to Computer Graphics](https://www.ua.pt/pt/uc/7930) course.

## Controls

- **WASD**: Move character
- **Mouse**: Move camera
- **Shift**: Run
- **E**: Interact
- **esc**: Unlock mouse
